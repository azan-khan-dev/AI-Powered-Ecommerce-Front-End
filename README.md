# AI-Powered Ecommerce Front-End

A modern, responsive e-commerce front-end built with **React** and **Tailwind CSS**.  
This project showcases a clean UI, dynamic product display, and smooth user interactions.

## Features
- Fully responsive layout for desktop, tablet, and mobile
- Interactive product listings with filters
- Auto-sliding hero banner
- Category and popular products sections
- User-friendly navigation and professional design
- Prepared for integration with backend APIs

## Tech Stack
- **React** – Front-end library for building interactive UI
- **Tailwind CSS** – Utility-first CSS framework for styling
- **Vite** – Development and build tool
- **ShadCN** – UI components for faster development
- **React Router** – For navigation and routing
